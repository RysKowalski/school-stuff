Listy tworzone za pomocą css

*Wszystkie wyświetlane listy mają color=red dla lepszej widoczności

Listy numerowane

list-style-type: lower-latin; - lista numerowana małymi literami alfabetu łacińskiego

<ol style="list-style-type: lower-latin;">
  <li>Punkt1</li>
  <li>Punkt2</li>
</ol>
  1. Punkt1
  2. Punkt2

list-style-type: armenian; - lista numerowana literami alfabetu ormańskiego

  <li>Punkt2</li>
</ol>
  1. Punkt1
  2. Punkt2

list-style-type: georgian; - lista numerowana literami alfabetu gruzińskiego

</ol>
