Listy tworzone za pomocą html

*Wszystkie wyświetlane listy mają color=red dla lepszej widoczności

Listy numerowane

<ol> - lista numerowana

<ol>
  <li>Punkt1</li>
  <li>Punkt2</li>
</ol>
  1. Punkt1
  2. Punkt2

Można także ustawić argument html start=liczba
Na przykład start=3

<ol start="3">
  <li>Punkt3</li>
  <li>Punkt4</li>
</ol>
  1. Punkt3
  2. Punkt4

<ol type="A"> - lista numerowana literami alfabetu

<ol type="A">
  <li>Punkt1</li>
  <li>Punkt2</li>
</ol>
  1. Punkt1
  2. Punkt2

<ol type="I"> - lista numerowana liczbami rzymskimi

<ol type="I">
  <li>Punkt1</li>
  <li>Punkt2</li>
</ol>
  1. Punkt1
  2. Punkt2

<ol type="i"> - lista numerowana małymi liczbami rzymskimi

<ol type="i">
  <li>Punkt1</li>
  <li>Punkt2</li>
</ol>
  1. Punkt1
  2. Punkt2

<ol type="a"> - lista numerowana małymi literami alfabetu
